# Citation Verification Checklist

**Why this document exists.** AI language models hallucinate academic citations at significant rates — inventing author combinations, wrong journals, plausible but fake page numbers, and real papers with incorrect years. Because some portion of the paper's literature review was AI-assisted, every citation in the bibliography must be verified manually against a primary source before publication.

**This document does not verify anything itself.** It formats the 15 references for efficient manual checking. Each entry includes a pre-constructed Google Scholar search string. Work through them one by one.

**What counts as verified.** For each citation:

1. Google Scholar or the publisher's website returns the paper with matching authors, year, title, and venue.
2. Volume, issue, and page numbers match.
3. A DOI (if available) resolves to the correct paper.

If any field is wrong, correct it in the paper's bibliography before publishing. If a citation cannot be found at all, remove it and replace the textual reference with a verified alternative.

---

## Process

For each entry below:

1. Copy the search string into https://scholar.google.com
2. Open the top result; confirm authors, year, title, journal, volume, issue, pages
3. Tick `[x]` on the `Verified` line if all fields match
4. If any field is wrong, note the correction next to `Needs correction:`
5. If the paper cannot be found at all, tick `Unable to locate — replace or remove`

## After verification

Once all 15 entries above are verified (or corrected):

1. Update the paper's bibliography with any corrections.
2. The README's AI disclosure currently says *"Citations were verified manually against primary sources."* That sentence only becomes true when this checklist is fully ticked. Do not publish the repo until it is.
3. Consider keeping this document in the repo (marked as "completed on DATE by NAME") as a permanent record of due diligence. It's a credibility signal for any reader evaluating the work's integrity.

## If a citation cannot be verified

Two options: replace with a verified alternative covering the same point, or remove the in-text citation and rewrite the sentence to not depend on it. Do not leave unverified citations in the paper. "Looks plausible but I can't find it" is not a safe state for any claim that a recruiter or peer might examine.
